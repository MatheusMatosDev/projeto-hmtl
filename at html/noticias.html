Remake de Resident Evil 3 é oficializado com trailer alucinante

A Capcom aproveitou o último State of Play do ano, o programa de anúncios da PlayStation, para oficializar o desenvolvimento do remake de Resident Evil 3 com um trailer alucinante! Assista logo acima.
O jogo ainda ganhou uma data de lançamento para 3 de abril de 2020.
Para a surpresa dos fãs, o jogo é a campanha misteriosa de Project Resistance, ou seja, o estúdio decidiu pegar os dois projetos e transformar em um só game.
Além disso, Project Resistance ganhou seu título oficial: Resident Evil Resistance.
